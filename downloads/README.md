# Downloads — FitMind

O APK Android de teste fica em `fitmind.apk` nesta pasta.

**Como instalar no Android**

1. Baixe `fitmind.apk` pelo card do projeto no portfólio ou em [GitHub Releases](https://github.com/cardoso-ix/fitmind/releases/latest).
2. Permita instalação de fontes desconhecidas (configurações do navegador ou do sistema).
3. Abra o arquivo e instale.
4. Crie uma conta de teste no app.

O APK é apenas para **Android**. No iPhone use Expo Go ou Safari conforme o [README do FitMind](https://github.com/cardoso-ix/fitmind).
